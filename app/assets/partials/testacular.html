If only the name was as awesome as it is. Be careful searching for this in Google..

Testacular is another test runner but when combined with PhantomJS we can fully automate all of our unit/integration tests in TeamCity without needing to go the URL in the browser. Just like any of the other test runners, Testacular will run through all of your tests but with its configuration capabilities, you can have it run against all the browsers you care about and you can run it against the headless browser, PhantomJS.
